# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added

- Initial release
- `Form` class for dynamic form generation
- `Storage` class for localStorage management
- `Table` class for data table rendering
- Support for multiple input types (text, email, tel, textarea, select, checkbox, radio, hidden)
- CRUD operations (Create, Read, Update, Delete)
- Callback system for form events
- Auto-generated IDs and timestamps
- Message notification system
- Custom attribute support
- ES6 module support

### Features

- Zero dependencies
- Lightweight and fast
- Browser localStorage integration
- Flexible configuration
- Event-driven architecture
